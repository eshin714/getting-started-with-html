THE BASICS OF THE WEB AND HTML

The WORLD WIDE WEB is made up of a collection HTML. You could say its a web of HTML. There are over 30 billion HTML links on the internet.

HTML stands for Hyper Text Markup Language which can be made up of 4 parts. The first is text content, which is what you see. The second is mark up, which is what it looks like. The third are references, which other documents such as images and videos. The last is links, which are "links" to other pages.
